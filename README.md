# channel-sale-bot

Бот для продажи подписок на закрытый канал. Делал под клиента, которому нужна была полная автоматизация - от оплаты до кика при истечении.

## Что внутри

- несколько тарифов (1 мес / 3 мес / год)
- оплата через ЮКасса (webhook, проверка IP)
- промокоды со скидкой и лимитом использований
- реферальная система с прогрессивным процентом (чем больше рефералов - тем выше %)
- вывод реферального баланса (с подтверждением от админа)
- онбординг при старте (шаги с фото/видео, настраивается из админки)
- автоуведомления за 3 дня и 1 день до конца подписки
- кик из канала при истечении
- рассылка по сегментам (все / купившие / не купившие)
- параллельная поддержка MAX (отдельный polling, та же БД)

## Стек

aiogram 3, aiohttp, aiosqlite, yookassa, python-dotenv

## Заметки

- бот проверяет подписки каждый час
- уведомления отправляются за 3 и 1 день до окончания

## Запуск

```
pip install -r requirements.txt
cp .env.example .env
python bot.py
```
